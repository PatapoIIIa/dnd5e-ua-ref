# Довідник з D&D 5e державною

Додаток із короткими правилами Dungeons & Dragons 5e.

**Вебсайт:** [https://5e.uapleplaying.com/](https://5e.uapeplaying.com/)

[Svelte](https://svelte.dev/) + [TailwindCSS](https://tailwindcss.com/) = ❤️

### Установка

```bash
npm install
```

### Розробка

```bash
npm run dev
```

### Компіляція

```bash
npm run build
```
